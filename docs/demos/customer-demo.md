# Customer Demo

## Objective

Demonstrate realistic customer support interactions from request to response or escalation.

## Prompts

1. What information do I need to check product warranty?
2. Can you check the status of service request SR-1001?
3. Can you check my repair status?
4. This is the third time the same issue happened and I need escalation.

## Expected Outcome

The flow is easy to explain, easy to run locally, and credible for enterprise architecture discussion.
