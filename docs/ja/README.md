# Smart Call Center Architecture Quiz

この教材は、Azure の 2 つのソリューションパターンを明確な要件に変換し、実装可能なアーキテクチャ図を学生に設計してもらうためのものです。

## 授業の流れ

1. [quiz.md](quiz.md) を学生に配布します。
2. 機能要件と非機能要件を満たすアーキテクチャ図を作成してもらいます。
3. [rubric.md](rubric.md) で採点します。
4. [reference-architecture.md](reference-architecture.md) を使って講師が解説します。
5. 最後に [demo-guide.md](demo-guide.md) に従ってローカルデモを実行します。

## 期待するアーキテクチャのテーマ

良い回答は、次の 2 つの流れを組み合わせます。

- リアルタイム音声エージェント: 電話連携、低遅延の音声対話、知識検索、CRM または有人エージェントへの引き継ぎ。
- 通話後分析: 録音または文字起こし、バッチ文字起こし、PII マスキング、要約、感情、意図、レポート。
