# Exercise 3 - Configure Application Settings

## Objective

Configure the application to connect to backend, Microsoft Foundry, and business action services.

## Configuration Examples

Use `config/workshop.env.example` as a reference.

## Tasks

1. Copy the sample env file to your local `.env`.
2. Fill in Azure, search, and Foundry values.
3. Confirm the app reads the correct backend and frontend URLs.
4. Confirm secrets are not hardcoded in source files.

## Validation

- [ ] Application settings load correctly
- [ ] App can send a message to the agent
- [ ] Response returns without config errors
