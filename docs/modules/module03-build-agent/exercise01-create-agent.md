# Exercise 1 - Create Customer Operations Agent

## Objective

Create a Microsoft Foundry agent for customer request automation.

## Agent Role

The agent acts as a customer operations assistant. It should:

- Understand customer requests
- Use enterprise knowledge when appropriate
- Ask clarifying questions when required
- Execute business actions only when sufficient information is available
- Escalate sensitive or unresolved issues

## Tasks

1. Create a new agent in Microsoft Foundry.
2. Assign the workshop model deployment.
3. Set the agent name and description for customer operations.
4. Save the agent ID for later modules.

## Validation

- [ ] Agent exists in the project
- [ ] Model deployment is attached
- [ ] Agent name and description match the workshop scenario
