# Exercise 1 - Create Customer Operations Agent

## Objective

Create an Azure AI Foundry Agent for customer request automation.

## Agent Role

The agent acts as a customer operations assistant. It should:

- Understand customer requests.
- Use enterprise knowledge where appropriate.
- Ask clarifying questions when required.
- Execute business actions only when sufficient information is available.
- Escalate sensitive or unresolved issues.

## Validation

You should have a basic agent that can respond to simple customer requests.
