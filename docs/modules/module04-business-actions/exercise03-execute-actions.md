# Exercise 3 - Execute Business Actions

## Objective

Test the agent with business action scenarios.

## Test Scenarios

1. Customer asks for service request status.
2. Customer provides a valid request ID.
3. Agent calls backend tool.
4. Agent summarizes the result.
5. Agent suggests next action if needed.

## Validation Checklist

- [ ] Tool call is triggered only when needed
- [ ] Required parameters are collected
- [ ] Response is customer-friendly
- [ ] Errors are handled gracefully
