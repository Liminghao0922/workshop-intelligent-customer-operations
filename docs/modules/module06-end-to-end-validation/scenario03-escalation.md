# Scenario 3 - Escalation

## Scenario

The customer has a complex or high-risk issue requiring human follow-up.

## Expected Flow

```text
Customer request
  ↓
Agent identifies escalation need
  ↓
Agent creates or recommends escalation
  ↓
Customer receives clear guidance
```

## Validation

- [ ] Agent does not overpromise
- [ ] Escalation criteria are followed
- [ ] Customer receives next-step guidance
- [ ] Handoff message is clear and respectful
- [ ] No unsupported resolution is claimed
