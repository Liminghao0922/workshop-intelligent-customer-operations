# Exercise 1 - Prepare Azure Environment

## Objective

Prepare the Azure environment used by the workshop.

## Tasks

1. Confirm Azure subscription access.
2. Create or select a resource group.
3. Confirm permissions for deploying application resources.
4. Record environment values in `config/workshop.env.example`.

## Validation

You should be able to identify the subscription, resource group, and region that will be used in later modules.
