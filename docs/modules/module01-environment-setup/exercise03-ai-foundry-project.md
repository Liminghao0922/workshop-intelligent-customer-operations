# Exercise 3 - Prepare Azure AI Foundry Project

## Objective

Prepare the Azure AI Foundry project used to create the customer operations agent.

## Tasks

1. Create or select an Azure AI Foundry project.
2. Confirm model deployment availability.
3. Prepare the agent workspace.
4. Record project endpoint and model identifiers.

## Validation

You should have a Foundry project ready for the agent module.
