# Demo Script

## Demo 1 - FAQ Request

Customer asks a product or warranty question. Agent retrieves answer from Fabric knowledge.

## Demo 2 - Service Request Status

Customer provides a service request ID. Agent calls backend API and summarizes status.

## Demo 3 - Escalation

Customer describes a complex issue. Agent identifies escalation need and provides next step.

## Closing Message

This architecture demonstrates how enterprise data and AI agents can work together to transform customer operations from manual triage into intelligent request resolution.
