# Demo Script

## Demo 1 - Grounded Voice Answer

Customer asks a product or warranty question. Agent retrieves answer from Fabric knowledge.

## Demo 2 - Resolved Call

Customer confirms the answer resolved the question. End the call and show that post-call analysis creates no Dynamics Case.

## Demo 3 - Unresolved Call

Customer reports a duplicate charge and requests follow-up. End the call, then show the queue event, structured analysis, and one Dynamics Case.

## Closing Message

This architecture demonstrates how grounded live assistance and deterministic post-call action can share one customer lifecycle without giving an agent direct write authority.
