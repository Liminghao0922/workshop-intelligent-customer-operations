# Smart Call Center 架构 Quiz

这份材料用于让学生把两个 Azure 解决方案模式转化成清晰需求，并设计一张可以落地实现的架构图。

## 教学流程

1. 把 [quiz.md](quiz.md) 发给学生。
2. 让学生画出满足功能和非功能需求的架构图。
3. 用 [rubric.md](rubric.md) 评分。
4. 用 [reference-architecture.md](reference-architecture.md) 做讲解和对照。
5. 最后按 [demo-guide.md](demo-guide.md) 演示本地运行效果。

## 期望架构主题

优秀答案应结合两条链路：

- 实时语音坐席：电话接入、低延迟语音对话、知识检索、CRM 或人工坐席转接。
- 通话后分析：录音或转写、批处理转写、PII 脱敏、摘要、情绪、意图和报表。
